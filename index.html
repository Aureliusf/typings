theme

type the theme code for example dracula in the text box then hit [windows: alt + t], [mac: cmd + ctrl + t], [linux: super + ctrl + t or alt + t] for every theme refer to the themes section

language

type the language code for example german in the text box then hit [windows: alt + l], [mac: cmd + ctrl + l], [linux: super + ctrl + l or alt + l] for every language refer to user guide

typing mode

type the mode code time or wordcount in the text box then hit [windows: alt + m], [mac: cmd + ctrl + m], [linux: super + ctrl + m or alt + m]

punctuation

type true or false then hit [windows: alt + p], [mac: cmd + ctrl + p], [linux: super + ctrl + p or alt + p] to activate/deactivate punctuations

for more info check the user guide
